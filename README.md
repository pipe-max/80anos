# Proyecto cth80-src

Proyecto frontend con Vite. Instrucciones rápidas:

- Instalar dependencias: `npm install`
- Desarrollo: `npm run dev`
- Build de producción: `npm run build`

Este repositorio está preparado para deploy automático en Netlify. Configura Netlify para que use el comando de build `npm run build` y el directorio `dist`.
